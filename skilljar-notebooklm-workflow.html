🗂️ 第一步：从 Skilljar 提取学习材料 EXTRACT

Skilljar 的课程内容大多是网页形式（文字 + 视频），NotebookLM 需要你喂入素材。以下三种方式按优先级排列：

直接粘贴网页链接

▶

NotebookLM 支持导入网页 URL。每学完一个 lesson，把页面 URL 加进去。这是最省力的方式。

⚠️ 注意：Skilljar 有些内容可能在登录墙后面，NotebookLM 抓不到，此时需要改用下面的方式。

手动复制文本

▶

如果 URL 导入失败，直接选中课程页面的文字内容，粘贴成 Google Doc 或纯文本文件，再导入 NotebookLM。

PDF 打印

▶

用浏览器的 打印 → 另存为 PDF 功能把每个 lesson 保存下来，然后上传到 NotebookLM。

✅ 保真度最高，推荐用于格式复杂或含图表的课程页面。

📓 第二步：在 NotebookLM 中组织 ORGANIZE

建议按课程创建独立的 Notebook，让每个 Notebook 的上下文集中，生成的摘要和问题质量更高。

Notebook 1
Building with the Claude API
API 完整知识体系
Notebook 2
Introduction to MCP
MCP 协议与进阶主题
Notebook 3
Claude Code in Action
Claude Code 工作流强化
💡 每个 Notebook 独立隔离，上下文集中，NotebookLM 生成的摘要、flashcard 和测验题质量显著更高。

🤖 第三步：用 NotebookLM 生成学习内容 GENERATE

导入素材后，可以在 NotebookLM 里使用以下功能：

  • 1 生成课程摘要和关键概念提炼，快速建立知识框架。
  • 2 要求生成 flashcard 格式问答，例如：给我生成 20 个 Q&A flashcard，涵盖这个 notebook 里的核心知识点
  • 3 让它生成测验题来检验理解深度，形成主动回忆练习。
  • 4 使用 Audio Overview 功能生成播客式讲解，通勤时收听，充分利用碎片时间。

🎯 建议优先学习的课程 PRIORITY

结合当前使用 Claude Code、搭建 MCP server、管理 Amazon agents 项目的场景，优先级排列如下：

优先级 课程 理由
P1 Claude Code in Action 直接强化现有的 Claude Code 工作流
P2 Introduction to Agent Skills 正在建 agents 系统，直接对口
P2 Introduction to Subagents 与 Agent Skills 配合使用
P3 Introduction to MCP + MCP Advanced Topics 已经在用 MCP，进阶知识很实用
P4 Building with the Claude API 巩固 API 使用的完整知识

⚡ 实操 Tip TIP

Skilljar 里有些课程包含代码示例和练习，光看 flashcard 不够。建议采用双轨并行策略：

概念复习轨
使用 NotebookLM 生成 flashcard、摘要和 Audio Overview 播客，强化记忆与理解。
代码实操轨
在 Claude Code 里实际跑一遍代码示例，通过动手实操加深理解，避免纸上谈兵。
✅ 两条线并行 — 概念记忆 + 动手实操，学习效果最佳。
