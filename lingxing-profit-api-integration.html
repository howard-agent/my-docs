背景 CONTEXT

JHFinance 财务看板已接入领星订单数据，但缺少平台级实际结算损益（含平台佣金、FBA费用、广告费、COGS等扣项）。需要通过领星 OpenAPI 利润报表端点获取 Amazon 每月真实毛利，同时调查其他平台（TikTok / Walmart / ML / Temu）是否有等价接口。

关键操作 6 STEPS

领星 API V2 签名与调用

▶

新增 _make_sign_v2（布尔值序列化为小写字符串，数组展开为 compact JSON）和 _call_v2（请求头增加 X-API-VERSION: 2）。

利润报表拉取

▶
