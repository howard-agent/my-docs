背景与目标 CONTEXT

财务看板已接入领星订单数据，但缺少平台级实际结算损益（含平台佣金、FBA 费用、广告费、COGS 等扣项）。需要通过领星 OpenAPI 利润报表端点获取 Amazon 每月真实毛利，同时调查其他平台（TikTok / Walmart / ML / Temu）是否有等价接口。

多平台利润报表 API 探测 404

测试 30+ 路径变体，TikTok / Walmart / ML / Temu 利润报表端点全部返回 404。

❌
结论：领星 OpenAPI 目前仅对 Amazon 开放利润报表接口，其他平台尚未支持。
决策：其他平台改用订单数据聚合方式进行简易毛利估算。

新增「各平台结算」Tab DONE

看板新增结算 Tab，位于「前7日」右侧，支持按月切换：

平台 数据来源 备注
Amazon 利润报表 API（真实 P&L） 含佣金、FBA、广告等全部扣项
TikTok 订单数据聚合（估算） 缺少平台扣费明细
Walmart 订单数据聚合（估算） 无 COGS，需后台录入
ML 订单数据聚合（估算） 按 50% 计入，无 COGS

平台顺序：TikTok → Amazon → Walmart → ML → Temu
